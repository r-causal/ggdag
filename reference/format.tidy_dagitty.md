# Format a `tidy_dagitty` object

Format a `tidy_dagitty` object

## Usage

``` r
# S3 method for class 'tidy_dagitty'
format(x, ..., n = NULL, width = NULL, n_extra = NULL)
```

## Arguments

- x:

  an object of class `tidy_dagitty`

- ...:

  optional arguments passed to format

- n:

  Number of rows to show

- width:

  Width of output

- n_extra:

  Number of extra columns to print

## Value

Character vector of formatted output
